# 개인정보 취급방침 (Privacy Policy)

## 크롬 즐겨찾기 정리 도구

최종 수정일: 2026-04-09

### 수집하는 정보

이 확장 프로그램은 개인 정보를 수집, 저장, 전송하지 않습니다.

### 데이터 처리

- 북마크 데이터는 사용자의 브라우저 내에서만 처리됩니다.
- 백업 데이터는 브라우저의 로컬 저장소(chrome.storage.local)에만 저장됩니다.
- 분석 결과는 외부 서버로 전송되지 않습니다.

### 네트워크 요청

- 죽은 링크 검사 기능 사용 시, 북마크에 저장된 URL로 HTTP 요청을 보냅니다.
- 이 요청에는 쿠키, 인증 정보 등 개인 식별 정보가 포함되지 않습니다.
- 링크 검사는 사용자가 직접 실행할 때만 작동합니다.

### 제3자 공유

어떠한 데이터도 제3자와 공유하지 않습니다.

### 문의

문의 사항은 GitHub 저장소의 Issues를 통해 연락해주세요.
