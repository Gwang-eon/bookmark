LOCAL TOOL

크롬 즐겨찾기 정리 도구

크롬 Bookmarks JSON 파일을 읽어서 죽은 링크, 중복 링크, 주제/도메인 분류, 중요도 점수를 한 번에 분석한다.

정리 기준
  • 죽은 링크
  • 중복 링크
  • 주제별
  • 도메인별
  • 중요도순

불러오기

파일 업로드 또는 로컬 경로를 직접 입력할 수 있다.

리눅스 기본 경로: ~/.config/google-chrome/Default/Bookmarks

맥 기본 경로: ~/Library/Application Support/Google/Chrome/Default/Bookmarks

윈도우 기본 경로: %LOCALAPPDATA%\Google\Chrome\User Data\Default\Bookmarks

대기 중

요약

현재 북마크 상태를 빠르게 볼 수 있는 핵심 지표다.

추천 조치

효과가 큰 정리 순서를 자동으로 제안한다.

상위 도메인

같은 도메인에 북마크가 몰린 곳부터 정리할 수 있다.

주제 분류

규칙 기반 분류라서 필요하면 분류 규칙을 확장하면 된다.

죽은 링크

시간 초과나 5xx는 의심 링크로 함께 표시한다.

중복 그룹

가장 중요도 점수가 높은 북마크를 대표로 남긴다.

전체 북마크

검색, 정렬, 상태 확인용 목록이다.

내보내기

죽은 링크와 중복을 제거한 뒤 HTML 또는 JSON으로 내려받을 수 있다.

직접 적용 · 롤백

원본 `Bookmarks` 파일은 적용 전 자동으로 gzip 백업된다.

안전하게 적용하려면 Chrome을 먼저 종료하는 편이 낫다.

백업 저장소: 불러오는 중
